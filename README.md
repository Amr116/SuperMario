# SuperMario
An attempt to recreate the first level of Super Mario game implemented in C++ using OpenGL.

## Prerequisites & DEPENDENCIES:
- freeglut lib 
- SDL lib

you can execute the install.sh to install the prerequisites
```
$sh install.sh
```
